github按star排名，作用讲解（一）

1:AFNetworking 网络请求库
2:SDWebImage 控件加载网络图片
3:ReactiveCocoa 函数式响应式编程
4:GPUImage 图像操作，滤镜等
5:Masonry 代码实现控件布局
6:MBProgressHUD 菊花加载指示器
7:RestKit 封装了请求数据、解析和存储（请求(使用AFNetworking)、模型字典转换和底层coredata操作）, REST时变现卓越
8:fmdb sqlite3数据库的操作封装
9:Mantle 对MVC中model层的封装，意在减少我们建立model的代码（唱吧6.0已使用）
10:MagicalRecord 使得操作cordate更简单
11:Alcatraz 管理插件的XCode插件
12:JSQMessagesViewController 聊天控制器，如果自己写聊天，还是可以使用的，减少很多代码
13:three20 iPhone应用开发框架,（自己定义了很多独特的控件/控制器：图片浏览器，短信编辑控制器等）
14:SVProgressHUD 菊花加载指示器
15: CocoaLumberjack 日志框架
16:iCarousel 卡片浏览库，很炫酷，同时需要懂高等数学
17:VVDocumenter-Xcode Xcode中快速生成注释
