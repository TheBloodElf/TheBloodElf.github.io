github按star排名，作用讲解（四）

github按照start排名，作用与名称：
91:FormatterKit 收集了精心构思的NSFormatter子类
92:iOS9AdaptationTips iOS 9 适配系列
93:nui 通过一个样式表对 iOS 应用进行外观定制，类似 CSS
94:XLForm 最灵活且最强大的创建动态表单的iOS库，同样创建tableview，这个可以节约9/10的时间（吹）
95:Aspects 一个简洁高效的用于使iOS支持AOP面向切面编程的库，它可以帮助你在不改变一个类或类实例的代码的前提下,有效更改类的行为
96:PKRevealController 左右侧滑菜单
97:SWRevealViewController 左右侧滑菜单
98:iRate 提醒用户去打分
99:NSLogger 是一款专门负责输出日志信息的工具,给高手使用，菜鸟不能发挥大作为
100:GSwipeTableCell 左右滑动cell出现按钮的库
101:CocoaHTTPServer 在mac os,app上内嵌http服务器
102:EGOTableViewPullRefresh 下拉刷新
103:SSKeychain 支持对存储在钥匙串中密码、账户进行访问，包括读取、删除和设置
104:FuzzyAutocompletePlugin xcode代码自动补全插件
105:VCTransitionsLibrary 控制器转场动画
106:appledoc 写代码时写好注释，然后可以通过这个工具生成文档，方便别人查看
107:terminal-notifier 一个命令行工具用来给Mac OS X的用户发送通知
108:Kiwi 写BDD(行为驱动开发)的插件，代码量少，功能强大
109:MacGap 为HTML/JS/CSS开发人员提供了一个Xcode项目来开发原生 OS X 应用。这些应用运行在 OS X 的 WebView 并利用了WebKit技术的优势。
110:iOS8-Sampler ios8的一些特性code例子
111:CRToast 用来简单创建出现在导航栏或者状态栏上的通知
112:QuickDialog 帮助开发者快速创建复杂的表单，实现包括登录界面在内的各种样式的TableView输入界面，此外，还可以创建带有多个文本域的表格及项目
113:MSDynamicsDrawerViewController 左右侧滑菜单
114:CBStoreHouseRefreshControl 自定义下拉刷新，通过plist文件(里面写的是开始和结束的点)你可以使用任何想要的形状（推荐使用这个设计出文字类的形状）
115:JBChartView 实现效果非常不错的折线统计图和柱状统计图。可以动态绘制图表。手指在图表上滑动，会显示对应的数值
116:Reader IOS PDF阅读器代码
117:EZAudio 根据音量实时显示波形图
118:origami 交互原型设计神器，Origami是一款Quartz Composer的插件，在安装QC之后安装Origami，会发现QC的界面变得更加友好，更加适合进行移动App的界面设计。
119:cheddar-ios 简单即时的任务管理器
120:JASidePanels 左右侧滑菜单
