github按star排名，作用讲解（五）

github按照start排名，作用与名称：
121:Objective-C 一些xcode中的代码片段
122:BeeFramework 是一个用HTML/XML+CSS技术开发原生App的整体解决方案
123:iOS7-Sampler ios7特性的一些例子
124:MessageDisplayKit 用于显示iPhone和iPad的消息页面，你可以发送文本、声音、图片、视频等消息。模仿微信App。
125:injectionforxcode 一个在虚拟机运行时，改变代码直接可以不重启虚拟机看到效果
126:iOS-boilerplate 这不是一个框架，是一个 iOS 应用程序的基础模板，但又包含一些相同的固件和一些广泛使用的第三方类库，这样你就不用费力气再造车轮。
127:idev-recipes www.idevrecipes.com开发代码库，有很多是讲解怎么创造轮子的
128:iOSInterviewQuestions iOS面试题集锦（附答案）
129:MKNetworkKit 一个网络请求库,吸取了ASIHttpRequest与AFNetWorking的优点，并加入了自己特有的功能。
130:shadowsocks-iOS 科学上网
131:libextobjc 一个非常强大的Objective-C库的扩展,
132:TMCache 数据存储，TMCache由两个本身相似的存储组成，一个是TMMemoryCache，存在于内存中，另外一个是TMDiskCache，存在于硬盘中
133:TeamTalk 蘑菇街TeamTalk源代码
134:MCSwipeTableViewCell tableviewcell左滑出现的内容自定义
135:Atlas-iOS 个轻量级，灵活的用户界面组件集。设计能够让开发人员快速和轻松集成原生聊天通信体验至他们的应用
136:ColorSense-for-Xcode 一款Xcode颜色插件，可让UIColor和NSColor更加可视化。输入就可以看到颜色
137:MMWormhole iOS扩展和其包含的应用间搭起通信桥梁，用于在不同的位置传递消息或来回共享命令。消息被归档到写入应用程序的共享App Group的文件。效果类似于应用和扩展之间的进程间通讯，虽然应用和进程之间不存在真正的通信。
138:Amethyst 平铺窗口管理器
139:YTKNetwork 猿题库封装的网络请求库，基于AFNetworking
140:FXForms 能够帮助开发者轻而易举地创建出基于表格的iOS表单
141:PaperFold-for-iOS 书页折页效果
142:EAIntroView 启动页
143:mogenerator core data命令行工具，快速生成文件
144:Knuff 一款开源的OS X应用，提供必要的信息就可以对某个注册设备发送推送，简化开发流程
145:TNavigationBar UINavigationBar的分类，使你可以动态的改变外观
146:M13ProgressSuite 进度条库（屏幕中间的）
147:ShiftIt 轻松用快捷键实现窗口管理
148:twui 一个支持硬件加速的 Mac 的 UI 框架：
149:REFrostedViewController 左右侧滑菜单，有蒙板效果
150:Sparkle 针对OS X的软件更新框架
