github按star排名，作用讲解（三）

61:JSPatch 使用JavaScript调用任何Objective-C的原生接口，获得脚本语言的能力：动态更新APP，替换项目原生代码修复bug。
62:Canvas 一个对原生动画的封装
63:YYText YYKit项目之一，富文本标签，图文混排等
64:KVOController 顾名思义，用于在于另外一个控制器一个属性改变时改变本控制器ui等（还是很强大的）
65:Bolts-ObjC Parse 和 Facebook 在内部使用的底层库, 方便移动开发，基于任务开发，把一个网络请求等操作写成一个任务（和rac差不多）
66:popping 基于Facebook Pop引擎的iOS动画库
67:ResearchKit 一个开源软件框架，可以很容易地创建用于医学研究或其他研究项目的应用程序。
68:TSMessages 非hub风格（屏幕中间显示）的提示信息展示，导航上面显示
69:TPKeyboardAvoiding 自动滚动，避免文本输入框被键盘遮罩
70:DateTools 一个用于处理日期和时间的 O-C 库
71:macdown 一个mac下的文本编辑器
72:XMPPFramework 使用Objective-C实现了XMPP协议，大大简化了基于XMPP的通信应用的开发
73:DTCoreText 图文混排
74:NewsBlur 新闻阅读器，很多人都称其为 Google Reader 的替代品, 这是它的源码
75:ECSlidingViewController iOS中一个ViewController的容器，这个将其中的子ViewController分成了二个层。二层view controller滑动，上面的不完全滑出，显示出后面的view controller。
76:SVPullToRefresh 下拉刷新控件
77:FXBlurView 模糊效果（毛玻璃）
78:Tweaks 应用调试工具，开发人员通过更改参数来调整动画时间、速度、颜色等。通过这种方法，设计师和工程师可以在自己的设备上使用它，而不会妨碍其他App测试。
79:KIF 开源的用户界面测试框架. 使用 KIF, 并利用 iOS中的辅助功能 API, 你将能够编写模拟用户输入，诸如点击，触摸和文本输入
80:iTerm2 OS X 下一款开源免费的的终端工具，可替代原生的 Terminal
81:XVim xcode插件，vim的编辑方式嵌入到xcode里面
82:cocos2d-objc Cocos2d for iOS and OS X, built using Objective-C
83:tapkulibrary 一些实用的控件组合，比如类似music的翻页效果，日历，alert view等
84:KSImageNamed-Xcode 输入图片名字就能浏览图片内容
85:FLAnimatedImage iOS的高性能GIF动画引擎
86:appirater 鼓励用户给app打分
87:iOS-Runtime-Headers iOS运行时头文件项目，可以得到私有api，并进行调用
88:json-framework JSON、模型互相转换
89:CSStickyHeaderFlowLayout 实现头部固定的CollectionView
90:Onboard 一个简单易用的 iOS 应用启动页，可显示多个启动页，可点击启动页的按钮。
