github按star排名，作用讲解（二）

github按照start排名，作用与名称：
18:MJRefresh 快速集成上拉刷新、下拉加载
19:realm-cocoa 数据库引擎，据说使用 Realm 比使用 SQLite 以及 Core Data 要快很多
20:FlatUIKit 扁平化ui（几乎没特殊效果），继承或对现有控件加分类完成，简单方便使用
21:FastImageCache 提升图片的加载和渲染速度
22:Shimmer 字体可发光
23:PNChart 带有动画效果的图表（折线、柱状、圆饼等）库
24:OpenEmu OS X原生游戏模拟器
25:BitBar lets you put the output from any script/program in your Mac OS X Menu Bar.（让你从你的mac菜单栏输出脚本）
26:YYKit 一组功能丰富的ios组件（字符串加密、数组随机乱序等）
27:CocoaAsyncSocket 这个是socket库，如果自己做聊天不用第三方（环信等），那么一般使用这个来做二次封装
28: SlackTextViewController 功能强大易用的TableView和CollectionView下的文字输入解决方案。具有文字输入框高度自适应，自动输入，复制单元格内容等诸多好用的特色。
29: FLEX iOS 开发的应用内调试和探测工具集。运行的时候，FLEX 提供一个浮动在应用之上的工具栏，用户可以查看和修改应用的每一个地方。
30:RESideMenu 界面侧滑，抽屉效果
31:nimbus iphone开发框架，比起three20更多效果
32:SWTableViewCell UITableViewCell左右滑动时出现按钮（取消关注，置顶等）
33:TTTAttributedLabel 基于UILabel，如果想要实现点击标签中的某一些文字响应自定义事件，用这个很方便
34:asi-http-request 网络请求库
35:slate 一个窗口管理程序
36:JSONKit JSON、型互相转换
37:Chameleon 扁平化iOS色彩框架
38:xctool iOS命令后编译、测试工具
39:WebViewJavascriptBridge JS和iOS交互
40:MWPhotoBrowser 图片大图浏览控制器
41:spectacle 轻松整理窗口，而无需使用鼠标。
42:facebook-ios-sdk 用于iOS应用程序与Facebook平台集成。
43:trip-to-iOS iOS学习资料网站
44:JazzHands UIKit一个简单的关键帧基础动画框架
45:Reachability 是一个iOS下检测设备网络环境用的库
46:JVFloatLabeledTextField 文本输入框，当有内容时占位文字出现在文本框上方（浮动式表单输入框）
47:PureLayout 添加约束，对原生NSLayoutConstraint分类
48:open-source-ios-apps Collaborative List of Open-Source iOS Apps(开源iOS apps合作名单)
49:MMDrawerController 抽屉效果
50:IQKeyboardManager 输入框随着键盘的弹出自动上下浮动等功能，继承简单aop思想
51:BlocksKit 以block方式做了很多有用的功能，比如：遍历数组等
52:JSONModel JSON、型互相转换
53:UITableView-FDTemplateLayoutCell 自动计算UITableView高度
54:PonyDebugger 优秀的iOS应用网络调试工具
55:SocketRocket WebSocket客户端库
56:GitUp Git界面
57:DZNEmptyDataSet 当表格视图/集合视图没有数据时，显示自定义页面
58:MJExtension JSON、型互相转换
59:AwesomeMenu 炫酷，放射型弹出菜单
60:ViewDeck 侧滑菜单
