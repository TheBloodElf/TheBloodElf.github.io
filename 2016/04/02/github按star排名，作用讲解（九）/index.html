github按star排名，作用讲解（九）

github按照start排名，作用与名称：
241:GrowingTextView 一个 UITextView 的扩展，实现了当文本增多和减少是对输入框本身的伸缩和滚动处理。
242:fb-mac-messenger fb的桌面管理
243:RETableViewManager 一个非常强大的使用数据驱动的 UITableView 内容管理。
244:ioscreator ioscreator.comd的教程
245:SCLAlertView 界面炫酷的alert view，支持block
246:ZFDragableModalTransition 自定义动画展示模态视图控制器，可从左边、右边或者底部推出模态视图。
247:hpple XML/HTML解析器
248:JTSImageViewController 一款有趣的图片浏览器。双击图片能够放大，将图片拨离画面就能够取消浏览。如果没有完全离开界面，图片能回到原位，并有回弹效果。用来做头像大图预览还不错
249:GroundControl 允许你远程配置你的app，服务器上的plist文件，
250:MGBoxKit 简单的渐变iOS表格，使用的代码量很少，自定义操作简单易懂，使用了先进的blocks设计模式。
包括基于手势操作，观察者、控制事件和自定义事件的blocks
251:Ono XML/HTML解析器
252:novocaine iOS和OSX应用开发中的音频框架CoreAudio功能强大，但复杂繁琐，使用起来非常困难，可以把你虐得伤痕累累、鲜血直流。于是就有程序员创造了 应用于OSX和iOS的高性能音频框架——novocaine。在CoreAudio框架下需要几天才能实现的功能，使用novocaine框架只需要几 行代码就能完成。
253:SXNews 高仿网易新闻
254:Fluid-for-Sketch Sketch插件，自动布局像约束
255:ChitChat 能创建一个使用点对点网络的聊天应用。p2p
256:FontAwesomeKit 为什么要使用FontAwesome呢,其实,它的字体就是矢量图,无论是放大还是缩小都不失真的矢量图哦.而且还可以做动画，可以取得字体的path
257:UI7Kit 低版本扁平化神器，让iOS7以下版本同样有扁平化效果UI7Kit
258:ODRefreshControl 下拉刷新
259:XcodeGhost XcodeGhost病毒资源
260:OAStackView 让stack view在ios7以上都能适用
261:SCRecorder 类似 Vine 的点击拍摄，动画过滤器，慢镜头，片段编辑相机引擎。
262:WYPopoverController 一个可非常方便进行定制的弹出控件。
263:TOMSMorphingLabel 可实现标签的文本和值之间的配置转换动画。触发这个动画就像设置一个标签的text属性一样简单。就是改变标签的内容，自带动画
264:selfcontrol 打开以后编辑屏蔽域名列表，一旦开始阻挡就是九十分钟，期间后悔重启机器和关闭程序都不能阻挡屏蔽的步伐。但是重装系统应该管用。
265:Typhoon 依赖注入
266:FDStackView 在ios6也能使用stack view
267:Pull-to-Refresh.Rentals-iOS 下拉刷新
268:ParseSourceCodeStudy Facebook开源的Parse源码分析【系列】
269:DOUAudioStreamer 基于 Core Audio 实现的用于 iOS/Mac 的流媒体播放引擎 DOUAudioStreamer。
270:socket.IO-objc Socket.IO是一个完全由JavaScript实现、基于Node.js、支持WebSocket的协议用于实时通信、跨平台的开源框架
