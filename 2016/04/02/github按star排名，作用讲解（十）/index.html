github按star排名，作用讲解（十）

github按照start排名，作用与名称：
271:iOS-Categories 一些有用的分类
272:JTCalendar 一个自定义的日历视图
273:DACircularProgress 一个圆环的进度指示栏。
274:IDMPhotoBrowser 图片浏览，滑动消失，图像加载
275:gitx 一个git的GUI专为Mac OS X
276:ios-snapshot-test-case iOS单元测试库
277:sequelpro os x上的MySQL数据管理
278:RBBAnimation block形式写动画
279:CYLTabBarController 最低只需传两个数组即可完成主流App框架搭建
280:MHVideoPhotoGallery 图片和视频管理控件。
281:MVVMReactiveCocoa 顾名思义mvvm和rac的组合使用
282:SDCycleScrollView “iOS第一图片轮播器” 无限循环图片轮播器
283:Core-Data-Editor 能帮助用户更方便的查看,编辑和分析应用数据。
284:AnimatedTransitionGallery 自定义动画转换画廊应用。很多转场效果
285:CargoBay StoreKit组件
286:SMPageControl 自定义分段控件，图片，间距，大小
287:PunchClock 一款基于 iBeacon 的签到软件，iBeacon（一个知道你离开某个地方的硬件，支持iOS）
288:ZSSRichTextEditor 适用于iOS的富文本WYSIWYG编辑器，支持语法高亮和源码查看。ZSSRichTextEditor包含所有WYSIWYG标准的编辑器工具。
289:specta TDD / BDD framework
290:XCDYouTubeKit YouTube video player for iOS, tvOS and OS X
291:Toast 模仿安卓的Toast，自定义弹出内容，可以是图片等
292:DesignerNewsApp Swift 开发的 DesignerNews 客户端，看着美美的！
293:RuntimeBrowser 运行时查看类方法、属性等的库
294:iVersion 动态监测版本是否有更新
295:DKNightVersion 为你的应用添加夜间模式
296:KNSemiModalViewController UIViewController+KNSemiModal 的组合，实现半模态的弹出视图。
297:IntentKit 一个简单的方法去管理第三方的url schemes
298:MSCollectionViewCalendarLayout UICollectionViewLayout的子类，有点像iOS日历效果
299:objc-TimesSquare 显示日历
300:Haneke 图像缓存库（sdwebimage也有这个功能）
