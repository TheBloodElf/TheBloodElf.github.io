github按star排名，作用讲解（十七）-md

github按照start排名，作用与名称：
481:MAObjCRuntime 运行时的包装
482:MMAppSwitcher iosa的切换视图换成自定义的，ios7后台切换一般是退出时候的截图，用了这个就是动态的了
483:BBUncrustifyPlugin-Xcode 代码规范，美化Xcode插件
484:ADTransitionController 实现多种视图切换动画。包括滑动、旋转、淡出、交换、回形针等。用于转场动画
485:AMSlideMenu 侧滑菜单
486:HAPaperViewController 受到Facebook Paper的启发，作出的一个布局转换功能的UICollectionView
487:HexFiend Mac OS X 的一个十六进制编辑器
488:RACollectionViewReorderableTripletLayout 集合视图的布局，拖动cell可以重新布局
489:AGGeometryKit-POP 一个结合AGGeometryKi（几何图形框架）t和POP的库
490:QBImagePicker 多选图片
491:TGCameraViewController 自定义的相机框架，轻松的集成到iOS项目中
492:limechat IRC（网络中继了聊天协议） Client for Mac ，irc的最大特点就是信息有一个统一的中转站
493:CodeExamples 一些代码例子
494:newsyc 一个iPhone的骇客新闻客户端
495:AYVibrantButton 是活力时尚的 iOS 8 按钮活跃效果，是 UIButton 的子类，包含一个简单优雅的外观，支持 iOS 8 的 UIVisualEffectView 和 UIVibrancyEffect 类，有毛玻璃和颜色渐变等效果
496:CHCSVParser 一个csv文件（CSV是一种通用的、相对简单的文件格式，被用户、商业和科学广泛应用）解析器
497:HGPageScrollView 书页的滚动视图（不是翻书效果），左右滑动，有点像safari的标签效果，这个效果自己都会写，没啥用
498:MongoHub-Mac MongoDB图形客户端
499:StreamingKit OSX and iOS通用的快速，可扩展的音频播放
500:SVPulsingAnnotationView 在地图中显示自己的位置的自定义的视图
501:MarqueeLabel 插入式的uilabel替换工具，能够动态的添加滚动，当字数长度超过给定的范围
502:DZNPhotoPickerController 一款照片搜索和选择器
503:meituan 高仿美团iOS版
504:RSKImageCropper 一款iOS图片裁剪类库，类似于联系人应用，通过移动图片，剪切圆形框所显示的图片。
505:AHKActionSheet 是 UIActionSheet 的替代品，提供基于 Block 的 API 和一个可定制的外观。
506:Lin xcode的插件，自动完成本地化
507:uebersicht 在这个世界上的任何一个地方观察你的机器发生的事情，http://tracesof.net/uebersicht/
508:ClusterPrePermissions 在系统询问用户前展示开发者自己的请求许可，从而减少了用户跳出应用进行设置的几率。同时应用可以再次选择一个合适的机会请求用户许可。
509:routable-ios 一个路由，全局的，你可以设置一个key-value，然后不用import就可以打开自己定的控制器（key-value中定义的）
510:TwitterCover 向下拖动ScrollView（TableView）时，ScrollView上方的图片会随着手指的拖动而放大并且变模糊。松开手指之后，图片随着ScrollView的回复原来位置而恢复原样
