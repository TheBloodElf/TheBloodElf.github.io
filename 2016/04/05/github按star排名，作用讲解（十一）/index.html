github按star排名，作用讲解（十一）

github按照start排名，作用与名称：
301:baker 基于 HTML5 的技术让你方便的创建可在 iPad 和 Kindle 上阅读的电子书。
302:KZBootstrap 一系列的脚本和注释，代码质量很差时，在编译时产生额外的错误和警告。
303:OpenUDID 是iOS禁止使用系统UDID之后的新解决方法
304:TWMessageBarManager 导航bar上显示通知信息
305:DLCImagePickerController ios不同的滤镜效果
306:chrome-cli 用命令行控制谷歌浏览器
307:DAKeyboardControl 键盘滑动消逝，就像系统内置的短信界面
308:CotEditor 轻量级的OS X文本编辑器
309:AudioStreamer 播放在线音频
310:nv mac os笔记应用
311:QuincyKit 崩溃日志管理
312:Valet 安全存储数据到Keychain，而不需要知道Keychain怎么工作的
313:Nocilla 测试网络请求的库（写Nocilla代码发送请求）
314:octokit.objc GitHub API client for Objective-C
315:LocationManager 轻松获取当前设备的位置
316:MGSplitViewController 模拟ipad的SplitView，但是功能不全
317:FCModel NSObject的一个扩展，让你方便的在数据库中操作自己的对象，基于FMDB 来做的
318:GraphSketcher 一个快速，简单的图形绘制和数据绘图应用程序（os x ipad）
319:canabalt-ios Canabalt应用的代码
320:ios-fontawesome fontawesome是一个图片字体（看着是图片，结果是一种字体，这样就不需要准备@2x@3x两种图片了），这个库是用过字体名称加载fontawesome对于的“图片”来显示
321:XYPieChart 简单方法绘制饼状图
322:RealReachability 实际网络连接监测（判断条件很多，但是其他库没有判断）
323:ASCIImage 用 NSString 和 ASCII 创建 UIImage / NSlmage 例子。（是点阵图）
324:kxmovie 视频播放器，基于ffmpeg
325:bilibili-mac-client bilibilimac客户端
326:OHAttributedLabel 一个开源的图文混排组件，利用这个开源库就很容易实现图文混排的效果，当然显示表情也就不在话下了
327:DropdownMenu 点击导航上的按钮，在导航bar下面出现菜单栏
328:SBShortcutMenuSimulator 虚拟机使用3d touch
329:TinderSimpleSwipeCards 可拖动的卡片控件（一叠卡片），向左右滑有不同的事件，可以自定义角度、力度等
330:Brushes 绘画app
