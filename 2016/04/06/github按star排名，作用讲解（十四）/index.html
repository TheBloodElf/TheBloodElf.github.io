github按star排名，作用讲解（十四）

github按照start排名，作用与名称：
391:MotionBlur 让你添加运动模糊效果（就是相当于东西快速跑动的残影）的iOS动画
392:vagrant-manager 方便的 Vagrant 机器管理工具,Vargrant是一种类似于VirtualBox的虚拟软件，它采用一种声明Ruby DSL的方法来描述你的环境(DSL：领域特定语言学习笔记 )
393:kxmenu 冒泡方式弹出垂直的菜单
394:objection 依赖注入，基于借口实现，减少耦合，不需要import就可以获得vc
395:Objective-C-RegEx-Categories 是NSRegularExpression的一个延展,它可以把Object-C中的很多正则表达式合并成一个,简化了代码
396:NHBalancedFlowLayout 瀑布流布局
397:PrettyKit ios各种风格样式uitableview
398:ios-image-filters 一组类似 PhotoShop（ps） 风格的 UIImage 图像过滤接口
399:BugshotKit 一款给开发者和测试者的 iOS 应用内 bug 报告工具，提供注释的截图和控制台日志。
400:NirZhihuDaily2.0 Swift精仿知乎日报iOS端
401:KYGooeyMenu 果冻效果的弹性动画弹出菜单
402:UzysAnimatedGifPullToRefresh 下拉刷新使用gif图片，只需要很少的代码
403:iOS7-day-by-day ios7的一些简单的工程实例
404:AMPopTip 动画弹出冒泡提示信息
405:SVSegmentedControl 各种样式的分段控件
406:SimFinger 一个假的虚拟机，可以向里面添加app，然后有个原点模拟你的手指，可点击
407:RNGridMenu 一个基于网格的带有动画效果的菜单视图，从中间弹出，具有背景模糊效果
408:VVeboTableViewDemo vvebo（用微博api模仿的应用）的表格视图
409:Briefs wireframes（ui原型图设计）应用的库
410:ensembles 异步的coredata库，扩展了系统的coredata，添加了mac os和ios的同步，可以同步icloud等存储服务器上的文件，该卡框架很容易扩展，包括服务器之间移动文件等服务
411:JTRevealSidebarDemo 侧滑菜单，是主和副视图一起平移的
412:ESJsonFormat-Xcode 将JSON格式化输出为模型的属性，一个xcode插件
413:UIDevice-with-UniqueIdentifier-for-iOS-5 用mac地址，来获取设备的唯一标识符
414:ESTMusicPlayer 优雅而简单的音乐播放器，具有完整的功能
415:cocoa-rest-client OS X app去测试HTTP/REST端点（REST是所有Web应用都应该遵守的架构设计指导原则。当然，REST并不是法律，违反了REST的指导原则，仍然能够实现应用的功能。但是违反了REST的指导原则，会付出很多代价，特别是对于大流量的网站而言。）
416:eigen 一款艺术方面的app源代码
417:PulsingHalo calayer的子集，用起来和uiview差不多
418:ObjectiveRecord 轻量级的使用core data，有点像Ruby on Rails
419:FBSimulatorControl 一个可以同时运行多个模拟器的辅助框架
420:FancyTabBar 一个简单的菜单按钮，有利于保持app界面的整洁和整齐，弹射效果
