github按star排名，作用讲解（十三）

github按照start排名，作用与名称：
361:DZNSegmentedControl 可替代UISegmentedControl展示相关数目，典型用法是用户资料部分。功能可调整控件的颜色、字体、大小以及动画持续时间。动画的，自适应的选择指示器。支持UIBarPositioning和UIAppearance
362:NSDate-TimeAgo NSDate的分类，上周，前30分钟，后30分钟等功能
363:ARAnalytics 集成了大部分的统计库，并且在这些库的上层又做了一层封装。它提供了稳定一致的API，我们调用这些API的时候，ARAnalytics底层会去调用我们实际使用的统计服务提供者（比如UMeng）的API。这样，当我们不想再继续使用当前的统计服务，而转向另一个统计平台的时候（比如从UMeng转到Flurry），我们大部分的统计代码是不需要更改的，这样就减少了开发者的精力。
364:KMCGeigerCounter 一个iOS帧速计算器，像盖革计数器那样，当动画丢失一帧时它就记录一次。掉帧通常是不可见的，但是很难区分55fps和60fps之间的不同，而KMCGeigerCounter可以让你观测到掉落5帧的情况。
365:PSCollectionView 类似于Pinterest的瀑布流效果
366:LXReorderableCollectionViewFlowLayout 集合视图布局，可以长按拖动cell到另外的位置然后自动再次布局，重新排序
367:CLImageEditor 提供了最基本的图片处理编辑功能。如编辑，滤镜，颜色调整，裁剪，旋转，色彩曲线功能。使用起来和自带的图片选择一样简单
368:HeapInspector-for-iOS 一个模拟了app中内存堆的iOS调试工具，通过该工具你可以直接在设备上发现app内存泄露等诸多问题，无需启动Instruments.
369:KRVideoPlayer 类似Weico（简单，轻松的微博客户端）的视频播放器，支持竖屏模式下全屏播放
370:Context-Menu.iOS 添加按钮（点击后出现几个垂直排列的按钮），可自定义icon，并可根据自己的喜好设计单元格和布局
371:YLGIFImage 播放gif图片，使用很少的内存
372:NWPusher 模拟推送（这样就不需要后台帮你了，自己搞推送）
373:CodePilot 一个 Xcode 5 插件，允许你不许使用鼠标就能快速地查找项目内的文件、方法和标识符。它使用模糊查询匹配（fuzzy query matching）去计算出一个根据相关性排序好结果列表；通过少许的按键你就可以跳到你想要查找的方法了。
374:iHasApp 打印出你安装过的所有app
375:ColorArt 类似iTunes的，取一幅图画的主要背景颜色等 (这是个Mac功能)
376:RATreeView iOS 上的树形控件,支持不限级数的树，可以用来做树状列表
377:ASValueTrackingSlider uislider的子类，加了冒泡显示当前的值效果
378:Interpreter node.js的编辑器，ios7以上可以使用（textview里面输入，然后编辑，有错误会弹窗提示）
379:CWStatusBarNotification 简单的方法让你能够在静态栏上显示通知信息
380:Form 非常灵活和强大的方式让你去建立表单
381:JVFloatingDrawer 抽屉效果，左右侧滑菜单
382:Kal 一个日历组件，当用户点击cell的时候出现当天相关的信息在下面
383:KLCPopup 内容自定义弹窗视图，alertview（有比较好看的动画）
384:jot 一个 iOS 框架,轻松添加图形和文字到图像中
385:ZCAnimatedLabel 动画替换标签内容，支持富文本，文字是满满跑到标签上面去的
386:Textual os x下的IRC（可以让你实时与其他人聊天）客户端
387:ReactiveViewModel mvvm和ran结合时通常会额外使用这个库
388:STPopup 提供了一个alertview（里面是导航视图，有些场景会用到）
389:articles objc.io的完整、准确、优雅的中文翻译版本
390:RZTransitions 动画切换视图（转场动画）
