github按star排名，作用讲解（十六）.md

github按照start排名，作用与名称：
451:expecta 一个开源判断是否相等的框架，expect(@”foo”).to.equal(@”foo”)；expect(foo).notTo.equal(1);expect([bar isBar]).to.equal(YES);expect(baz).to.equal(3.14159);
452:PHFComposeBarView iOS 7 Messages.app 的输入框组件bar
453:BCGenieEffect mac下面菜单栏菜单的弹出和收回效果，吸入动画
454:Coding-iOS Coding iOS 客户端源代码
455:card.io-iOS-SDK 让card.io（扫描银行卡，获取银行卡号）更快、简单的实现在手机端
456: FSCalendar 日历控件
457:UISS 相当于css的方式去给控件加属性
458:AMSmoothAlert 动画弹出alertview
459:CoconutKit 从简单的视图控制器到先进的本地化功能，CoconutKit可以减少iOS开发者对样板代码（Boilerplate code）的编写，提高代码的质量和执行可靠的应用程序框架，这样，开发者就可以有更多的精力和时间来设计应用程序。
460:TDBadgedCell UITableViewCell 的子类实现了在列表项的右侧显示徽章信息
461:DFImageManager 高性能、异步、线程安全的图片加载、进度、缓存库
462:EGOCache key-value本地缓存，TMCache比这个好用
463:YTKKeyValueStore 猿题库公司的key-value本地缓存
464:DBCamera 简单的自定义的相机
465:APAddressBook 简单的连接address book，不需要太多的代码
466:RGCardViewLayout 卡片浏览器
467:LNPopupController 弹出视图，效果类似apple music
468:Rebel cocoa框架用于加强appkit（mac开发）
469:libPhoneNumber-iOS 标准的电话号码库（google的电话号码处理库）
470:CorePhotoBroswerVC 快速集成高性能照片浏览器，支持本地及网络相册
471:calloutview 一个轻量级的标注视图，模仿UICalloutView（google map上出现过）
472:Sol 一款天气应用，你可以自己计划自己的日程
473:superdb 通过命令行无限调试你的app
474:TBIconTransitionKit POP动画的按钮内容改变，只有四种
475:CBZSplashView 推特的启动动画，一个图变大然后消失，出现应用界面
476:ios-custom-alertview 自定义弹出视图，可以加图片和视图
477:GRMustache html文件的渲染工具
478:KeepingYouAwake 是mac的一个工具，让你的mac不会进入休眠状态
479:RMPZoomTransitionAnimator 自定义缩放过渡动画控制器，你点击某一cell的时候放大显示详情，推出时缩放到你点击的位置
480:scifihifi-iphone Open source iPhone code
